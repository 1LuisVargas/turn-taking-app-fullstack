# User Stories

1. As guest/anonymous, I want to be able to register for a new account.
    - Fill a form with email and password.
2. As a registered user, I want to be able to login.
    - Fill a form with email and password to login.
3. As guest/anonymous, I shouldn't be able to login.
4. As an already authenticated user, I want to be able to request for an appointment date.
    - Fill a form specifying the date and time for the appointment.
5. As an already authenticated user, I want to be able to check my historical appointments.
    - Filter appointments through a menu.
6. As an authenticated user, I want to be able to cancel my future appointments up to 24 hours in advance.
7. As an authenticated user, I want to modify the information in my account.

## Extra Credit

1. As a registering user, I want to confirm that the account has been created by a confirmation email.
2. As a registered user, I want to be able to upload a profile picture.
